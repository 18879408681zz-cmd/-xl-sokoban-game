# 如何分享推箱子游戏给朋友

## 问题
使用 `file:///C:/Users/xl188/Desktop/sokoban-game/index.html` 这样的地址只能在本地电脑上打开，无法在手机上或其他人的电脑上访问。

## 解决方案
使用本地HTTP服务器，让同一网络下的设备都能访问。

## 方法一：使用Python（最简单）

### 步骤：
1. 确保电脑安装了Python（如果没有，请从 https://www.python.org 下载安装）
2. 打开命令提示符（按 Win+R，输入 cmd，回车）
3. 切换到游戏目录：
   ```
   cd Desktop\sokoban-game
   ```
4. 启动HTTP服务器：
   ```
   python -m http.server 8000
   ```
5. 服务器启动后，你会看到类似这样的信息：
   ```
   Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
   ```

### 在手机上访问：
1. 确保手机和电脑连接**同一个Wi-Fi网络**
2. 查找电脑的IP地址：
   - 在命令提示符输入 `ipconfig`
   - 找到"无线局域网适配器 WLAN"下的"IPv4 地址"，例如：`192.168.1.100`
3. 在手机浏览器输入：`http://电脑IP地址:8000`
   - 例如：`http://192.168.1.100:8000`

## 方法二：使用批处理文件（双击运行）

我已经创建了 `start-server.bat` 文件，你可以：
1. 双击 `start-server.bat` 文件
2. 按照提示操作

## 方法三：使用Node.js（如果你有安装）

1. 安装Node.js（如果没有）
2. 在游戏目录打开命令提示符
3. 安装简单HTTP服务器：`npm install -g http-server`
4. 运行：`http-server -p 8000`

## 注意事项

1. **防火墙**：第一次运行时，Windows可能会询问是否允许访问，请选择"允许"
2. **网络**：确保所有设备在同一网络下
3. **电脑不能休眠**：服务器运行时，电脑需要保持开机状态
4. **端口占用**：如果8000端口被占用，可以改用其他端口（如8080）

## 停止服务器
在命令提示符窗口中按 `Ctrl+C` 即可停止服务器。

## 高级选项：部署到互联网
如果想让任何人随时随地都能访问，可以考虑：
1. GitHub Pages（免费）
2. Vercel/Netlify（免费）
3. 购买域名和虚拟主机
